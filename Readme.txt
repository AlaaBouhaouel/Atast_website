to run this website locally :
0.make sure you downloaded all the libraries and dependencies
1.open cmd 
2.navigate to the folder that holds the manage.py file
3.run the following command:
python manage.py runserver 0.0.0.0:8000

4.You will see such a message in normal cases :
System check identified no issues (0 silenced).
May 21, 2024 - 12:39:57
Django version 5.0.6, using settings 'atast.settings'
Starting development server at http://0.0.0.0:8000/
Quit the server with CTRL-BREAK.

you have to copy the following and run it in your browser : http://<LAPTOP_IP adress>:8000/


Note: in case this website is not publically hosted yet you can run it on any device connected 
to the same wifi after following these steps:

1.change the device's IP address in the seconde position in the 28th line of code in settings.py 
2.Make sure ur device is connected to the same wifi as your laptop (where the server is running)
3.open your devices browser and type 'http://<LAPTOP_IP adress>:8000/'
4.you are supposed to see the website running in your device 
else check the wifi.
